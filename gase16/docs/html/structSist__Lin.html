Descrição detalhada

Essa estrutura anônima foi definida para o tipo Sist_Lin, que representa o sistema linear resultante da discretização da equação diferencial.

As variáveis dInfx, dInfy, dPrin, dSupx e dSupy representam os valores das diagonais não-nulas da matriz A do sistema. Os inteiros nx e ny são o número de pontos em cada dimensão, passados como argumento para o programa.
