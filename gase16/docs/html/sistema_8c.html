Essa função aloca o vetor solução u(x,y) e o inicializa. Esse vetor representa a região do plano para a qual se quer resolver a equação diferencial. O vetor é preenchido com zeros nos seus pontos internos e com as condições de contorno nas bordas da região. No caso da equação do trabalho a região e as cndições eram: O dominio D = [0, PI] x [0, PI], onde: u(x,0) = sin( 2*PI*(PI-x) ) * sinh( PI*PI ) u(x,PI) = sin( 2*PI*x ) * sinh( PI*PI ) u(0,y) = 0 u(PI,y) = 0.
