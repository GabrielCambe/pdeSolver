◆ aloca_e_inicializa_solucao()

void aloca_e_inicializa_solucao ( Real_t **  u,
Sist_Lin *  sist 
)

Essa função aloca o vetor solução u(x,y) e o inicializa. Esse vetor representa a região do plano para a qual se quer resolver a equação diferencial. O vetor é preenchido com zeros nos seus pontos internos e com as condições de contorno nas bordas da região. No caso da equação do trabalho a região e as cndições eram: O dominio D = [0, PI] x [0, PI], onde: u(x,0) = sin( 2*PI*(PI-x) ) * sinh( PI*PI ) u(x,PI) = sin( 2*PI*x ) * sinh( PI*PI ) u(0,y) = 0 u(PI,y) = 0.

Parâmetros
sistPonteiro para o sistema que representa a equação diferencial.
uVetor solução do sistema.

Essa função escreve em um arquivo os valores calculados pelo método de Gauss-Siedel. Os valores de u(x,y) são postos junto ao seus respectivos x e y num arquivo do tipo grid do gnuplot. Também são escritos os comentários acerca da execução do programa, o resíduo por iteração e o tempo médio de execução do método.

Parâmetros

num_iterUm inteiro com o número de iterações feitas na execução de gaussSiedel.
residuo_iterUm vetor de tipo Real_t, os resíduos para cada iteração.
